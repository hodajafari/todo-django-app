# Django To-Do App

A simple task management web application built with Django.

Users can create, edit, delete, and manage tasks.

---

## Features

- User registration
- Login and logout
- Create tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Search tasks
- REST API for tasks

---

## Technologies

- Python
- Django
- Django REST Framework
- HTML
- CSS
- Bootstrap
- SQLite
- Git

---

## Installation

Clone the repository

git clone https://github.com/Hodajafari/todo-django-app.git

Go to project folder

cd todo-django-app

Create virtual environment

python -m venv .venv

Activate environment

.venv\Scripts\activate

Install packages

pip install -r requirements.txt

Run migrations

python manage.py migrate

Run server

python manage.py runserver

---
